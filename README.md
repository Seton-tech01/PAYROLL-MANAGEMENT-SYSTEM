# Payroll Management System

A payroll management system developed using **VB.NET** to automate employee salary processing, payroll calculations, and record management.

## Overview

This application was designed to simplify payroll administration by automating salary computations, deductions, and employee payroll record tracking.

It reduces manual errors and improves payroll processing efficiency.

## Features

* Employee registration and management
* Salary structure setup
* Automatic payroll calculation
* Allowance management
* Deduction management
* Net salary computation
* Payroll history tracking
* Search and filter records
* User-friendly interface

## Technologies Used

* VB.NET
* Windows Forms
* MySQL / SQL Server
* ADO.NET

## Modules

### Employee Management

Manage employee details including:

* Employee ID
* Name
* Department
* Position
* Salary information

### Payroll Processing

Handles:

* Basic salary calculation
* Allowance computation
* Tax/deduction processing
* Net salary generation

### Reports

Generate payroll reports and salary summaries.

## Purpose

This project demonstrates practical payroll automation using VB.NET and database integration.
